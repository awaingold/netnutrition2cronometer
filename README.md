This is an unpackaged browser extension. To use, navigate to chrome://extensions, enable developer mode in the top-right corner, then click "add unpacked." Double click on the NetNutrition2Cronometer
folder to install the extension.

Usage:
1. Open NetNutrition and navigate to the nutrition information for the desired dish.
2. Open a Cronometer tab and navigate to the interface for creating a new food.
3. Open the extension and click "import nutrition"
